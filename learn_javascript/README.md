### 第一章 JavaScript 简介
- 基本上就是简单的介绍，直接跳过，对于一个写了 13 年 JavaScript 的人来说，如果这些基础的基本知识都没掌握，那就真的白干了。
- JavaScript 的变量类型：数字、字符串、布尔值、函数和对象。还有 undefined 和 null，以及数组、日期和正则表达式
- ECMAScript 是一种脚本规范，JavaScript就是这个规范的一个实现，jscript和ActionScript也是如此
- ECMAScript 6 的功能
    - let 和 const
    - 模版字面变量
    - 解构
    - 展开操作符
    - 箭头函数: =>
    - 类
